# AGENTS.md

## Overview

awesome-ai-practices-list is a curated collection of real-world AI best practices from X, auto-updated every 6 hours by DigestOps.

## Build & Test

No build step. Content is Markdown files organized by topic.

## Code Style

- One practice per entry with title, summary, source link, and date
- Entries sorted by date (newest first) within each topic file

## Boundaries

- Content auto-generated by DigestOps pipeline
- Manual edits welcome for corrections and categorization
- Do not add entries without source attribution
